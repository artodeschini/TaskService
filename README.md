# TaskService
A Simple RestFull Sample Quarkus

## In this sample I use:

* Java 11
* Maven 3.8.4
* Quarkus
* JPA / Panache with Postgres
* JSON
* jQuery (only use to bootstrap)
* Bootstrap 5
* HTML5

## Build

```bash
mvn clean install
```

```bash
mvn clean install
java -jar target/tasks-2.0.0.jar
```

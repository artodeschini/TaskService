# TaskService
A Simple RestFull Sample Quarkus

## In this sample I use:

* Java 11
* Maven 3.8.4
* Quarkus
* JPA / Panache with h2 Databases
* JSON
* jQuery (only use to bootstrap)
* Bootstrap 5
* HTML5

## to build

```bash
mvn clean install
```

## to run application

```bash
java -jar target/quarkus-app/quarkus-run.jar
```
